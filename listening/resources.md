# JLPT N1 Listening Resources (1-Month Sprint)

## Primary Practice
- JLPT Official Practice (新しい問題例) Listening — use as baseline; redo weekly.
- 日本語の森 N1 Listening playlists — explainers with scripts.
- NHK News Web Easy (audio) — shadow daily article, then transcript check.
- NHKラジオ ニュース (regular speed) — 5 min headlines, build real exam speed.

## Shadowing Sets
- 「耳から覚える」語彙N1 — short dialogues; focus on intonation.
- JLPT Sensei Listening Mock Tests — timed sections to simulate pressure.
- 「新完全マスター聴解N1」 — if available, schedule 2 units per week.

## Strategy Tips
1. **先読み**：設問・選択肢は音声前に把握し、キー語彙にマーク。
2. **段階調整**：Easy→Normal→Fastの順でシャドーイング、毎回録音して自分の音を確認。
3. **トランスクリプト分析**：聞き取れなかった表現は語彙CSVへ追記、音読→意味定着。
4. **設問分類**：意見一致／事実確認／意図把握…毎回メモし弱点を可視化。

## Rapid Drills (10–15 min options)
- 「にほんごコンテッペイ」上級 — casual topics, good for warm-up。
- Satori Reader (Audio) — adopt narration, pause to predict content。
- Podcast「ひいきびいき」 — fast native talk; use 0.75x then ramp up。

Stick to 30–40 min focused listening daily (split AM/PM) plus shadowing.
