# Grammar Additional Exercises (N1-focused)

This file expands the `exercises_01.md` set with translation, error correction, and sentence production tasks.

## Section A — Fill-in (選択)
Complete the blanks with the most appropriate pattern.

1) 事態は深刻(    )、速やかな対処が必要だ。［に至って／にあって／を問わず］
2) 彼の説明は理屈(    )、納得できない。［にかなって／に反して／に耐えない］
3) あの計画は見直す(    )、実行は見送られた。［に至らず／までもなく／にあって］

## Section B — Error recognition
Identify and correct the misuse of grammar patterns in the following sentences.

1) 例：彼は若いにもかかわらず、落ち着いていた。(correct)
2) その提案は簡単に採用すべきではない、利点もあるが、欠陥もあるに至る。(incorrect — correct it)
